HTML(HyperText Markup Language)은 웹 페이지의 구조와 내용을 정의하는 마크업 언어입니다. HTML은 웹 문서를 구성하는 기본 요소로, 텍스트, 이미지, 링크, 폼 등의 다양한 콘텐츠를 표시합니다. 태그를 사용해 요소를 정의하며, <html>, <head>, <body>와 같은 기본 태그를 포함합니다. <h1>부터 <h6>까지의 헤딩 태그, <p> 태그로 단락을, <a> 태그로 하이퍼링크를, <img> 태그로 이미지를 삽입합니다. HTML 문서는 브라우저에 의해 해석되어 사용자가 볼 수 있는 웹 페이지로 렌더링되며, CSS와 JavaScript와 함께 사용되어 웹 페이지의 스타일과 동작을 제어합니다.
