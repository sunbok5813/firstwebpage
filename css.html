CSS(Cascading Style Sheets)은 웹 페이지의 외형을 정의하는 스타일 시트 언어입니다. HTML이 구조와 콘텐츠를 담당한다면, CSS는 글꼴, 색상, 여백, 배경, 레이아웃 등 시각적인 요소를 지정합니다. CSS는 선택자(selector)를 사용해 특정 HTML 요소를 선택하고, 해당 요소의 스타일 속성(property)과 값을 지정하여 외형을 조정합니다. 또한, CSS는 계층적(Cascading) 특성을 가지므로 스타일 규칙이 겹칠 때 우선순위가 적용됩니다. 이를 통해 웹 디자이너와 개발자는 일관된 디자인을 유지하면서도 다양한 장치와 화면 크기에 대응하는 반응형 웹 디자인을 구현할 수 있습니다. CSS는 내부 스타일(HTML 내부에 작성), 외부 스타일(별도의 CSS 파일에 작성), 인라인 스타일(HTML 요소 내에 직접 작성) 등 다양한 방식으로 적용할 수 있습니다.
